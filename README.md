# Screen Monitor

**Screen Monitor** — это десктопное приложение для мониторинга заданной области экрана и автоматического обнаружения новых аварийных ситуаций в системах БТИ. Приложение анализирует цвет пикселей (#A0A0FF) и при значительном увеличении его площади подаёт звуковой сигнал, отправляет уведомление в Telegram и/или показывает всплывающее окно в Windows.

---

## Основные возможности

- **Выбор области мониторинга** – интерактивное выделение мышью на всех подключённых мониторах.
- **Настраиваемый интервал проверки** – от 1 до 60 секунд.
- **Звуковые оповещения** – для каждого региона свой звуковой файл (Волга, Юг, Северо‑Запад, Центр, Москва).
- **Telegram‑уведомления** – отправка сообщений через собственного бота при обнаружении новой аварии.
- **Всплывающие Windows‑уведомления** – мигающее окно поверх всех приложений.
- **Поддержка нескольких мониторов** – корректная работа с любыми конфигурациями экранов.
- **Сохранение настроек** – все параметры хранятся в файле `.env`.
- **Современный интерфейс** – тёмная тема, скруглённые углы, адаптация под высокий DPI.

## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/ldixitl/screen_monitor.git
cd screen-monitor
```

### 2. Создание виртуального окружения (рекомендуется)
```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка окружения
Создайте файл `.env` в корневой папке проекта. Пример содержимого:
```ini
# Основные настройки
INTERVAL=10
REGION=Волга
SOUND_VOLUME=50
AUDIO_DEVICE=                 # оставьте пустым для устройства по умолчанию

# Telegram (необходимо создать своего бота)
TELEGRAM_ENABLED=False
TELEGRAM_CHAT_ID=
TELEGRAM_BOT_TOKEN=           # вставьте токен вашего бота

# Windows-уведомления
WINDOWS_NOTIFICATIONS_ENABLED=True

# Координаты области мониторинга
MONITOR_LEFT=11
MONITOR_TOP=233
MONITOR_WIDTH=1976
MONITOR_HEIGHT=220
```
*Если файл не существует, программа создаст его автоматически с настройками по умолчанию.*

### 5. Запуск приложения
```bash
python main.py
```

## Настройка Telegram‑бота

Для получения уведомлений в Telegram необходимо:

1. Найти в Telegram [@BotFather](https://t.me/botfather) и создать нового бота командой `/newbot`. Следуйте инструкциям – вы получите токен вида `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`.
2. Вставить полученный токен в файл `.env` в поле `TELEGRAM_BOT_TOKEN`.
3. Узнать свой Telegram ID.
4. Включить Telegram‑уведомления в интерфейсе программы и ввести свой ID в соответствующее поле. Можно также сразу нажать «Тест Telegram» для проверки.

## Использование

1. **Выберите область мониторинга** – нажмите «Выбрать область» и мышью выделите нужный участок экрана (обычно область списка аварий). Появится затемнённый фон и размеры выделения.
2. **Настройте интервал проверки** – сколько секунд ждать между сканированиями.
3. **Выберите регион** – для каждого региона можно использовать свой звук оповещения (файлы должны лежать в `resources/sounds/` с именами `volga_alarm.mp3`, `south_alarm.mp3` и т.д.).
4. **Включите нужные типы уведомлений** – Telegram и/или всплывающие окна.
5. **Сохраните настройки** (кнопка «Сохранить настройки»).
6. **Нажмите «Запустить мониторинг»** – при обнаружении новой аварии вы услышите звук, увидите запись в журнале и, если настроено, получите сообщение в Telegram.
