# Screen Monitor

**Screen Monitor** — это десктопное приложение для мониторинга заданной области экрана и автоматического обнаружения новых аварийных ситуаций в системах БТИ. Приложение анализирует цвет пикселей (`#A0A0FF`) и при значительном увеличении его площади подаёт звуковой сигнал, отправляет уведомление в Telegram и/или показывает всплывающее окно в Windows.

---

## Основные возможности

- **Выбор области мониторинга** — интерактивное выделение мышью на всех подключённых мониторах.
- **Настраиваемый интервал проверки** — от 1 до 60 секунд.
- **Звуковые оповещения** — для каждого региона используется свой встроенный звуковой файл.
- **Пользовательские звуки** — можно выбрать собственный звук для каждого региона через интерфейс программы.
- **Telegram-уведомления** — отправка сообщений через бота при обнаружении новой аварии.
- **Всплывающие Windows-уведомления** — мигающее окно поверх всех приложений.
- **Поддержка нескольких мониторов** — корректная работа с любыми конфигурациями экранов.
- **Поддержка высокого DPI** — корректная работа при масштабировании интерфейса Windows.
- **Сохранение настроек** — все параметры хранятся в файле `.env`.
- **Современный интерфейс** — тёмная тема, скруглённые углы и кастомные окна.

## Звуковые файлы регионов

Встроенные звуки расположены в папке `resources/sounds/`:

- `volga_alarm.mp3` — Волга
- `south_alarm.mp3` — Юг
- `north_alarm.mp3` — Северо-Запад
- `centre_alarm.mp3` — Центр
- `mimo_alarm.mp3` — Москва
- `east_alarm.mp3` — Дальний Восток
- `siberia_alarm.mp3` — Сибирь
- `ural_alarm.mp3` — Урал

Пользовательские звуки, выбранные через интерфейс программы, копируются в папку:

```text
resources/user_sounds/
```

Это позволяет сохранить работоспособность настроек даже в том случае, если исходный аудиофайл был удалён или перемещён пользователем.

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/ldixitl/screen_monitor.git
cd screen-monitor
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка окружения

Создайте файл `.env` в корневой папке проекта. Пример содержимого:

```ini
# Основные настройки
INTERVAL=10
REGION=Волга
SOUND_VOLUME=50
AUDIO_DEVICE=

# Telegram
TELEGRAM_ENABLED=False
TELEGRAM_CHAT_ID=
TELEGRAM_BOT_TOKEN=

# Windows-уведомления
WINDOWS_NOTIFICATIONS_ENABLED=True

# Координаты области мониторинга
MONITOR_LEFT=11
MONITOR_TOP=233
MONITOR_WIDTH=1976
MONITOR_HEIGHT=220

# Пользовательские звуки по регионам
CUSTOM_SOUND_VOLGA=
CUSTOM_SOUND_SOUTH=
CUSTOM_SOUND_NORTHWEST=
CUSTOM_SOUND_CENTER=
CUSTOM_SOUND_MIMO=
CUSTOM_SOUND_EAST=
CUSTOM_SOUND_SIBERIA=
CUSTOM_SOUND_URAL=
```

Если `.env` отсутствует, программа создаст его автоматически со значениями по умолчанию.

### 5. Запуск приложения

```bash
python main.py
```

## Настройка Telegram

Для получения уведомлений в Telegram необходимо:

1. Найти в Telegram [@BotFather](https://t.me/botfather) и создать нового бота с помощью команды `/newbot`.
2. Получить токен бота.
3. Указать токен в `.env` в параметре `TELEGRAM_BOT_TOKEN`.
4. Узнать свой Telegram ID.
5. Включить Telegram-уведомления в интерфейсе программы и ввести свой ID.
6. При необходимости воспользоваться кнопкой **«Тест Telegram»**.

## Использование

1. **Выберите область мониторинга** — нажмите **«Выбрать область»** и выделите мышью нужный участок экрана.
2. **Настройте интервал проверки** — задайте период между сканированиями.
3. **Выберите регион** — для каждого региона используется свой встроенный или пользовательский звук.
4. **При необходимости настройте звук** — можно выбрать устройство вывода, изменить громкость и назначить собственный аудиофайл.
5. **Включите нужные типы уведомлений** — Telegram и/или всплывающие окна.
6. **Сохраните настройки**.
7. **Нажмите «Запустить мониторинг»**.

При обнаружении новой аварии приложение:
- воспроизводит звуковой сигнал;
- добавляет запись в журнал;
- показывает всплывающее уведомление;
- отправляет сообщение в Telegram, если эта функция включена.
